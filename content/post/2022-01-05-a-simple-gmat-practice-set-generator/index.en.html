I recently sat for the GMAT. To practice smarter, I came up with a simple Official Guide 2020 problem set generator in Python. The motivation came from my dissatisfaction with available solutions, for example, the solution you get by registering the book online. Some don’t allow you to narrow the problems down by sub-category, others aren’t random generators or don’t keep track of done problems. On the other hand, using my simple generator, you can still input the answers in any online system and thus take advantage of any statistics offered.

The code and more details at: https://github.com/andleb/set_generator
